# ZeroLine UI – Frontend Task

## Overview

This project is a frontend implementation of the **ZeroLine Home Page UI** based on the design provided in the assignment.
The objective was to recreate the layout accurately using clean HTML and CSS while maintaining proper structure and responsiveness.

The project focuses on layout alignment, spacing, typography, and component positioning to match the provided UI as closely as possible.

---

## Features

* Clean and structured HTML layout
* Styled navigation bar with logo and menu
* Divider placement under the navigation bar
* Arrow navigation section
* Grid-based content layout
* Consistent spacing and typography
* Responsive design using media queries

---

## Technologies Used

* **HTML5**
* **CSS3**
* **Flexbox**
* **CSS Grid**
* **Media Queries**
